# Proiect Identificarea Sistemelor

Identificarea Sistemelor Dinamice (Ordin II)

Acest proiect explorează diverse metode de identificare a modelelor matematice pentru sisteme dinamice, utilizând un set de date realizat experimental și prelucrarea acestora în MATLAB. Lucrarea este structurată pe trei piloni principali de analiză: identificarea aperiodică (regresie liniară), analiza în frecvență (metode neparametrice) și metodele parametrice. De asemenea, s-a făcut manual generarea unui semnal SPAB potrivit pentru sistemul în cauză ca să fie excitat destul încât dinamica sistemului sa fie surprinsă. Pentru răspuns în frecvență a fost calibrat un semnal VCO (Voltage-controlled oscillator) pentru a surprinde dinamica sistemului.
